## Introduction 📖

Using data from 2023 to 2025, this project presents key insights into various data roles to help job seekers evaluate their career options. It enables users to assess the potential of each role by analyzing metrics such as salary, work schedule (e.g., full-time, part-time), and other relevant factors, supporting more informed career decisions.

## Skills Showcased 🛠️
In order to build this project, we implement several Power BI features. Here's what we mastered:

**🔧 Data Preparation (ETL) with Power Query:** Transformed raw data into an analysis-ready format by handling missing values, adjusting data types, and generating calculated columns.

**🧠 Implicit Measures:** Developed measures to extract key insights and KPIs, such as Median Annual Salary and Total Job Count.

**📈 Core Visualizations:** Applied Column, Bar, Line, and Area charts to compare job volumes and analyze trends over time.

**🌍 Geospatial Insights:** Used map visuals to represent the global distribution of job opportunities.

**🔢 KPIs & Tabular Views:** Implemented Cards to highlight key metrics and Tables to present detailed, sortable data.

**🎨 Dashboard Design:** Crafted a clean and user-friendly layout, combining standard and advanced visuals to effectively communicate insights.

**🖱️ Interactive Features:**

🎛️ **1. Slicers:** Enabled dynamic filtering by Job Title.

🔘 **2. Buttons & Bookmarks:** Built smooth and intuitive navigation across the report.

🔍 **3. Drill-Through:** Allowed transition from summary-level views to deeper, context-rich details.

## Dashboard Overview 🧩    
The report is structured across two pages to provide a more detailed and comprehensive analysis.

### Page 1: Job Market Landscape   
![Job selected](/Images/Drill%20Through%201st%20page.png)
This dashboard provides a high-level overview of the data job market, showcasing core KPIs—total job count, median salaries, and top job titles—for quick insights at a glance.

### Page 2: Job Role Drill Through
![Job drill through](/Images/Drill%20Through%202nd%20page.png)
This page delivers a focused deep dive into a specific job title. Using drill-through from the main dashboard, you can analyze salary ranges, work-from-home metrics, top job platforms, and global job locations.

## Conclusion 🎯
This Data Jobs Dashboard highlights the power of Power BI in transforming raw job market data into actionable insights. By combining data transformation, advanced metrics, geospatial visuals, and interactive features, it provides a clear and dynamic view of the data jobs landscape.

The dashboard:

- 📊 Synthesizes 2023–2025 data into clear, intuitive visualizations
- 🎛️ Enables interactive exploration through slicers and drill-through
- 🔢 Highlights key metrics like salary and employment type
- 🧭 Supports informed career decision-making

Overall, this project demonstrates strong BI fundamentals and the impact of effective, user-focused dashboard design.
