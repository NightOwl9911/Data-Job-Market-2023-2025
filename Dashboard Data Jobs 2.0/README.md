## Introduction 📖

Using data from 2023 to 2025, this project builds on Project 1 by providing deeper insights into data roles. It analyzes key metrics such as hourly and annual salaries, along with the most in-demand skills for each role, enabling more precise and informed career decisions.

## Skills Showcased 🛠️
To build this project, we leveraged several Power BI features. Here’s what we mastered:

🖌️ **Dashboard Design:** Developing a clean, intuitive, and visually engaging report layout.

🔧 **Power Query (ETL):** Executing data cleaning, transformation, and structuring processes.

🔗 **Data Modeling:** Designing optimized data models using relational structures (Star Schema concepts).

🧠 **DAX Fundamentals:** Implementing calculations and aggregations to uncover meaningful insights.

📊 **Visualizations Applied:**

📉 **1. Core Charts:** Column, Bar, Line, and Area visuals for comparisons and trend analysis.

🌐 **2. Map Visuals:** Representing geographic data distribution.

🔢 **3. KPI Cards:** Showcasing key metrics at a glance.

📄 **4. Tables:** Displaying detailed, structured datasets.

🎭 **5. Chart Diversity:** Combining standard and advanced visuals for stronger data storytelling.

🖱️ **Interactivity Features:**

🎛️ **1. Slicers:** Allowing dynamic, user-controlled filtering.

🔘 **2. Buttons & Bookmarks:** Enhancing navigation and managing report states, including Drill-Through functionality.

## Conclusion 💡
Data Jobs Dashboard 2.0 delivers a streamlined, single-page experience that combines clarity with analytical depth. It centralizes key metrics, expands salary insights (hourly and annual), highlights in-demand skills, and enables dynamic exploration through intuitive slicers.

By balancing simplicity with functionality, this project demonstrates strong Power BI fundamentals—efficient data modeling, clean ETL, and effective DAX—while supporting faster, more informed decision-making.

Overall, it showcases how thoughtful design can transform complex data into clear, actionable insights.
