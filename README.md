<h1 align="center"> 💼 Data Job Market Analysis: Skills, Salaries & Trends </h1>

I've developed two Power BI dashboards to analyze data job roles, required skills, and salary trends. The project leverages a dataset and guidance from Luke Barousse’s Analytics course.

### 1. Data Jobs Dashboard version 1 📊
![Dashboard v1](/Images/Dashboard%20v1.png)

Interactive Power BI dashboard analyzing key metrics across various data roles, using multiple visualizations (charts, tables, and trends) to highlight job demand, salary insights, and role characteristics. Designed for individuals exploring or transitioning into data careers.
